# Preview Routes Context

Rotas dentro de `src/app/preview` sao internas para alinhamento visual do design system.

## Regras

- Nao trate `/preview` como pagina publica do portfolio.
- Use para demonstrar componentes reutilizaveis, variantes e interacoes importantes.
- Mantenha exemplos pequenos e objetivos.
- Nao crie previews para toda secao simples da home; priorize componentes com variantes ou comportamento especifico.

## Cores

- Use tokens de `globals.css` sempre que possivel.
- Evite introduzir novas paletas ou estilos que nao existam na home.
