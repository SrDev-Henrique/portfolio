# Preview Components Context

Esta pasta contem componentes usados para a vitrine interna `/preview`.

## Objetivo

- Ajudar outros agentes e o usuario a visualizar componentes reutilizaveis e variantes.
- Manter exemplos objetivos, em fundo escuro, sem virar uma pagina publica.

## Padrao

- Nome do componente.
- Descricao curta.
- Demonstracao real.
- Quando houver variantes, mostrar cada uma com label e props relevantes.

## Cores

- Previews também devem respeitar os tokens do `globals.css`.
- Se o componente principal migrar para tokens globais, atualize a preview junto.
